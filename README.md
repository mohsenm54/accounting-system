# 📊 Accounting System for Moddian Platform

یک سامانه حسابداری جامع و یکپارچه برای سامانه مودیان

## 🎯 ویژگی‌های اصلی

- ✅ مدیریت حساب‌های شناور
- ✅ دفتر روزنامه و دفتر بزرگ
- ✅ تهیه صورت‌حساب‌های مالی
- ✅ گزارش‌های تفصیلی
- ✅ مدیریت کاربران و سطوح دسترسی
- ✅ یکپارچگی با سامانه مودیان

## 🏗️ معماری پروژه

```
accounting-system/
├── backend/              # Node.js + Express Server
├── frontend/             # React Application
├── database/             # Database Scripts
└── docs/                 # Documentation
```

## 📦 تکنولوژی‌های استفاده شده

### Backend
- **Node.js** + **Express.js**
- **PostgreSQL** - پایگاه داده
- **JWT** - احراز هویت
- **Joi** - اعتبارسنجی داده‌ها

### Frontend
- **React.js**
- **Axios** - درخواست HTTP
- **React Router** - مسیریابی
- **CSS Grid** - طراحی واکنش‌پذیر

## 🚀 شروع سریع

### پیش‌نیازها
- Node.js v14+
- PostgreSQL 12+
- npm یا yarn

### نصب و اجرا

```bash
# Backend
cd backend
npm install
npm start

# Frontend (در Terminal جدید)
cd frontend
npm install
npm start
```
